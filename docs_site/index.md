# Pharos Engine

*Pharos Engine* is a 2D pixel-art game engine with an optional 3D
layer, a notebook-diary editor, a Rust `_core` extension, and a
wgpu / VCR render backend.

## Two wheels

| Wheel | pip install | Purpose |
| --- | --- | --- |
| `pharos-engine` | `pip install pharos-engine` | Headless engine + Rust `_core`. |
| `pharos-editor` | `pip install pharos-editor` | Notebook editor UI (pulls in `pharos-engine`). |
